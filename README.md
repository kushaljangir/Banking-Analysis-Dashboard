# 🏦 Banking Analysis Dashboard

## 🎯 Project Objective

To analyze banking customer data and identify risk levels, profitability, and loan trends using Power BI.

## 🛠 Tools Used

* Power BI
* DAX
* Excel

## 📊 Key Features

✔ Customer Segmentation
✔ Risk Score Analysis
✔ Loan Approval Trends
✔ Profitability Analysis
✔ Default Rate Tracking

## 📈 Key Insights

* Identified high-risk customers
* Found most profitable customer segments
* Analyzed loan approval patterns
* Detected regions with high default rates

## 📂 Dataset

Contains 1500+ customer records with loan, balance, and risk details.
